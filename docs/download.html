Download (Linux)

Recommended: download the latest Linux x86_64 tarball from GitHub Releases.

Go to latest release

Install

# 1) download dwf-linux-x86_64.tar.gz from Releases
tar -xzf dwf-linux-x86_64.tar.gz

# 2) move binary into your PATH
sudo install -m 0755 dwf /usr/local/bin/dwf

# 3) verify
dwf --help

Use in a Rust project

cd /path/to/your-rust-project
dwf init
dwf run fast
dwf run full
dwf report --last 10
Advanced users can verify downloads using the accompanying .sha256 file in each release.
