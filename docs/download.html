Download (Linux)

Recommended: download the latest Linux x86_64 tarball from GitHub Releases.

Go to latest release

Install

# 1) download dwf-linux-x86_64.tar.gz from Releases
tar -xzf dwf-linux-x86_64.tar.gz

# 2) move binary into your PATH
sudo install -m 0755 dwf /usr/local/bin/dwf

# 3) verify
dwf --help

Use in a Rust project

cd /path/to/your-rust-project
dwf init
dwf run fast
dwf run full
dwf report --last 10

Case Study — Sample Results

The following results were generated using dwf report --last 10 on a small Rust project (echo_rs). They illustrate how dwf surfaces actionable workflow metrics. 


dwf report --last 10

Report (last 10 requested, found 3):
  ok: 2 | failed: 1
  TTS  avg: 6321 ms | median: 8154 ms
  TTG  avg: 9348 ms | median: 9348 ms
  Failure stages:
    fmt      1

Interpretation:

  • One formatting failure was caught almost instantly — demonstrating the value of the fail-fast gate.
  • Median TTS ≈ 8.1s, reflecting typical developer feedback time.
  • Median TTG ≈ 9.3s, driven primarily by test execution.
  • These results support a workflow of frequent dwf run fast and occasional dwf run full.
