Commands

dwf init            # create dwf.toml
dwf run fast        # fmt → check → clippy (fail-fast)
dwf run full        # fast + tests
dwf report --last 10
dwf doctor

KPIs

TTS (Time-to-Signal): time until the first failure or until success if none failed.

TTG (Time-to-Green): time until the whole pipeline is green (only exists on success).

Pipeline rationale

Cheap checks first: formatting takes milliseconds; compilation and tests can take seconds. This minimizes wasted time and shortens iteration cycles.
