Ship faster by shortening your feedback loop.

dwf is a Rust CLI that runs a fail-fast pipeline (fmt → check → clippy → test), records timings, and reports KPIs like TTS (Time-to-Signal) and TTG (Time-to-Green).

Download for Linux Read docs

Fail fast

Stops at the first failure so you don’t waste time compiling/testing when formatting already fails.

Measure what matters

Logs per-stage timings and produces reports so you can optimize the right bottleneck.

CI-friendly

Non-zero exit on failure. Use it locally or in GitHub Actions.

Example

$ dwf run fast
Mode: Fast | ok: true
✅ fmt     257 ms
✅ check   5658 ms
✅ clippy  4627 ms
TTS: 10542 ms
TTG: 10542 ms

Community Edition — Free (default)

The fully open-source CLI you have built.

  • dwf run fast — staged, fail-fast pipeline
  • dwf run full — full validation with tests
  • dwf report — local TTS/TTG analytics
  • dwf doctor — workflow diagnostics
  • JSON history in .dwf/history.jsonl
  • Linux binaries via GitHub Releases
  • Works locally and in basic CI

Best for: students, individual developers, and small Rust projects.
